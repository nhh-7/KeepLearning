	mv：移动和重命名文件和目录
	rm：删除文件和目录
	ln：创建硬链接和符号链接

##### 创建目录
	mkdir dir1
	也可同时创建多个目录
	mkdir dir1 dir2 dir3

##### 复制文件和目录
	cp item1 item2 将单个文件或目录item1复制到文件或目录item2
	cp item... directory 将多个文件或目录复制到目录directory

##### 移动和重命名文件
	mv item1 item2 将文件或目录item1移动或重命名为item2 
	mv item... directory 将一个或多个item从一个目录移动到另一个目录

##### 删除文件和目录
	rm item... 删除一个或多个文件或目录

##### 创建硬链接和符号链接
	硬链接: ln file link
	硬链接不能引用目录,只能引用文件
	
	符号链接: ln -s item link 其中item可以是文件或目录
	符号链接是一种特殊类型的文件,其中包含指向目标文件或目录的文本指针.
	创建符号链接中的item是根据link所在的位置而形成的相对位置(当然也可以用绝对位置)
	
![[Pasted image 20240918152813.png]]
	
	如上图中ln -s ../fun dir1/fun-sym 创建的符号链接fun-sym在dir1目录下面,而链接的文件fun与dir1同级,所以要使用../fun
