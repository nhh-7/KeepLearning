##### read-从标准输入读取值
	read可用于从标准输入中读取一行。使用重定向也可以读取文件的数据行。
	read [-options] [variable...]
	variable是一个或多个变量，用于保存输入值。如果未指定变量，则输入值保存在Shell变量REPLY中。

###### IFS
	Shell通常会对提供给read的输入进行单词分割。也就是输入行中被一个或多个空白字符分隔的多个单词会变成若干个独立项，再由read分配给各个变量。Shell变量的 IFS 控制此行为，IFS的默认值包含了空格符、制表符、换行符。修改IFS，就可以提取由其他字符分割的多个项。

	
	![[Pasted image 20241010151456.png]]
	该脚本 提示输入 用户名，然后显示该用户再/etc/passwd中对应的各个字段。
	
	IFS=":" read user pw uid gid name home shell <<< "$file_info"
	这条语句中，对IFS赋值出现在read命令之前，这意味着这个赋值的效果时临时的，只对后面紧跟着的命令有效。执行完read命令后，IFS的值就恢复到原来。
	<<< 表示here string。将file_info的值传到read命令的标准输入中。
	注意：read命令不能放入管道使用。
