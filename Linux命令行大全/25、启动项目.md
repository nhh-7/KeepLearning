###### 为变量与常量赋值
	variable=value
	注意：变量名、赋值号、值这三者之间绝不能出现空白字符。

	在进行参数扩展时（即 $变量名 ），变量名两边可以有选择地添加花括号。如果变量名由于其所在的上下文而产生歧义，花括号就能发挥作用了。

###### here document
	here document是另一种I/O重定向形式，我们利用它在脚本内嵌入文本块并将其送至命令的标准输入。
		command << token
		text
		token
	其中token可以是任意一个字符串，用于标示嵌入文本的起止。token字符串必须单独出现，所在行的末尾绝不能由空白字符。
	
	![[Pasted image 20241009211415.png]]
	here document与echo大同小异 ，只不过默认情况下，here document中的单引号和双引号不具备特殊含义。将其作为普通字符对待。

	如果将 << 改为 <<-, Shell就会忽略 here document中每行文本开头的制表符（但不忽略空格符），这样就能对text进行缩进。
