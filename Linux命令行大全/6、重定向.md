	cat：拼接文件
	sort：排序文本行
	uniq：报告或忽略重复的行
	wc：统计文件中换行符、单词以及字节的数量
	grep：输出与模式匹配的行
	head：输出文件的开头部分
	tail：输出文件的结尾部分
	tee：读取标准输入并将输出结果写入标准输出和文件

I/O重定向允许我们修改输出结果的去处和输入的来源。通常，输出结果显示在屏幕，输入则来自键盘，有了I/O重定向，两者皆可改变。

##### 标准输出重定向
	使用重定向操作符>，后面跟上文件名，即可将标准输出重定向到其他文件。
	ls -l /usr/bin > ls-output.txt
	使用重定向符>对标准输出进行重定向时，会完全重写目标文件。
	> filename 使用此命令可以清空一个文件的内容，或者创建一个新文件

	要将重定向的标准输出追加到文件尾部，而不是从头覆盖，可使用 >> 实现
	ls -l /usr/bin >> ls-output.txt
	使用 >> 操作符会将标准输出追加到文件尾部。

##### 标准错误重定向
	想重定向标准错误，必须引用其文件描述符。
	标准输入、标准输出、标准错误 在 Shell内部分别用文件描述符0、1、2引用它们的。
	ls -l /bin/usr 2> ls-error.txt

	2> 表示对标准错误进行重定向，正常输出则不会就进行这个重定向 。
	
###### 将标准输出和标准错误重定向到同一个文件中
	使用 &> 操作符
	ls -l /bin/usr &> ls-output.txt
	使用 &>> 进行追加
###### 丢弃用不着的输出结果
	将输出结果重定向到名为/dev/nul的特殊文件
	该文件可以接受输入结果但不做任何处理
	ls -l /bin/usr 2> /dev/null

##### 标准输入重定向
	拼接文件
	cat filename 可以显示文件内容
	cat命令可以接收多个文件作为参数，所以可用于将文件拼接到一个文件中。
	cat file1 file2 > filename

	如果没有为cat命令指定任何参数，他就从标准输入中读取。
	cat > filename
	在这条命令后输入想放入文件的文本，最后按Ctrl+D 表示已到文件尾。就可以将输入的内容放到该文件中。

	使用<操作符进行 重定向输入
	cat < filename
	重定向输入为文件输入

##### 管道
	通过管道操作符 | ，可以将一个命令的标准输出传给另一个命令的标准输入：
	command1 | command2
######  排序列表
	ls /bin /usr/bin | sort | less
###### 报告或忽略重复行
	uniq命令可以从标准输入或单个文件名参数中获取有序的数据列表，默认删除所有重复行。
	ls /bin /usr/bin | sort | uniq | less
###### 统计文件中换行符、单词、字节的数量
	wc ls-output.txt
	ls /bin /usr/bin | sort | uniq | wc -l  使用 -l 选项限制只输出文件行数
###### 输出与模式匹配的行
	grep pattern filename
	-i 使grep命令在搜索时忽略字母大小写
	-v 使grep命令只输出不匹配指定模式的行
###### 输出文件的开头/结尾部分
	head/tail filename 默认输出文件的前十行和后十行，行数可通过 -n 控制
###### 读取标准输入并将输出结果写入标准输出和文件
	tee命令从标准输入读取内容，然后将其复制到标准输出和 其他文件中。
