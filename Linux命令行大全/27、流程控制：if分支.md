##### if语句
	if commands; then
		commands
	[elif commands; then
		commands...]
	[else
		commands]
	fi
	其中commands是命令列表
##### 退出状态
	命令（包括脚本和Shell函数）在结束时会向系统返回一个值，这就是退出状态-是一个取值范围在0-255的整数。数值0表示执行成功，其他数值表示失败。
	
	![[Pasted image 20241009215550.png]]
	可以使用？参数来检测退出状态。

	当if之后是命令列表，则判断其中最后一个命令的退出状态值。
	
	![[Pasted image 20241009215736.png]]
##### 使用test
	test命令执行各种检查和比较
	两种形式：
		test expression
		[ expression ]
	其中 expression是一个表达式，求值结果为真那么test命令返回退出状态值0，如果为假，则返回1。

##### 更现代的test
	[  [expression]  ]
	其中expression是一个表达式，结果要么为真，要么为假。
	[[]]命令支持所有test表达式，且额外支持 string =~ regex
	如果string匹配 ERE regex，则返回真。
	
	![[Pasted image 20241010104914.png]]

	[ [] ]的另一个特性是其中 == 操作符支持和路径名扩展一样的模式匹配。
	
	![[Pasted image 20241010105317.png]]

	(())可用于执行算术真值测试，如果算术求值的结果不为0，则测试结果为真

##### 控制操作符：另一种分支方式
	使用操作符 && 和 ||
	对于command1 && command2，先执行command1，仅当command1执行成功时才执行command2
	对于command1 || command2，先执行command1，仅当command1执行失败时才执行command2
