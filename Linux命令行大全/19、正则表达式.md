	正则表达式是一种用于识别文本模式的符号表示法。
##### grep
	grep（global regular expression print）全局正则表达式输出。基本功能是在文本文件中搜索与指定的正则表达式匹配的文本，将含匹配项的文本行输出到标准输出。
	grep [options] regex [file...]，其中regx代表正则表达式

##### 元字符与文字字符
	元字符用于指定复杂的匹配，包括 ^ $ . [ ] - ? * + ( ) | \
	其他所有字符被视为普通字符，反斜线可用于创建元序列，还能转义元字符，使其成为普通字符。
	当包含元字符的正则表达式出现在命令行上时，要将其放入引号中，避免Shell去扩展这些字符。

##### POSIX基本型正则表达式与扩展型正则表达式
	基本型正则表达式（BRE）与扩展型正则表达式（ERE）之间的不同是元字符不同。
	BRE识别以下元字符： ^ $ . [ ] *
	ERE又加入了以下元字符： ( ) { } ? + |
	如果使用反斜线将 ( ) { } 转义的话，BRE视其为元字符；而ERE会将转义后的字符视为文字字符。
	grep程序可以使用-E选项来支持ERE
