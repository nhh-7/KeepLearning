### 管理用户
	MySQL用户账号和信息存储在名为mysql的MySQL数据库中。

##### 创建用户账号
	CREATE USER hhn IDENTIFIED BY '123'
	# 重命名用户账号
	RENAME USER hhn to fym;
##### 删除账号
	DROP USER fym;
##### 设置访问权限
	# 显示用户的权限
	SHOW GRANTS FOR hhn;
	
	使用GRANT语句设置权限。 GRANT要求你至少给出以下信息：
	 要授予的权限；
	 被授予访问权限的数据库或表；
	 用户名
	
	#授予权限
	GRANT SELECT ON crashcourse.* TO hhn;
	
	#撤销权限
	REVOKE SELECT ON crashcourse.* FROM hhn;
	
	GRANT和REVOKE可在几个层次上控制访问权限：
	 整个服务器，使用GRANT ALL和REVOKE ALL；
	 整个数据库，使用ON database.*；
	 特定的表，使用ON database.table；
	 特定的列；
	 特定的存储过程。

	简化多次授权 可通过列出各权限并用逗号分隔，将多条GRANT语句串在一起，如下所示：
	GRANT SELECT, INSERT ON crashcourse.* TO hhn;

##### 更改口令
	SET PASSWORD FOR hhn = PASSWORD('123456');
	SET PASSWORD更新用户口令。新口令必须传递到Password()函数进行加密。
	
	SET PASSWORD还可以用来设置你自己的口令：
	SET PASSWORD = PASSWORD('123456');
	在不指定用户名时， SET PASSWORD更新当前登录用户的口令。
