使用SELECT语句的ORDER BY子句，根据需要排序检索出的数据。

#### 排序数据
	SELECT prod_name FROM	products ORDER BY prod_price;
	根据ORDER BY后跟的列名进行排序

#### 按多个列进行排序
	SELECT prod_id, prod_price, prod_name FROM products ORDER BY prod_price, prod_name;
	只有prod_price相同，才会进一步根据prod_name进行排序

#### 指定排序方向
	SELECT prod_id, prod_price, prod_name FROM products ORDER BY prod_price DESC;
	不指定排序方向则为默认升序ASC，使用DESC为降序

	SELECT prod_id, prod_price, prod_name FROM products ORDER BY prod_price DESC, prod_name;
	DESC关键字只应用到直接位于其前面的列名，上面的语句意思为按照prod_price进行降序排序，如果prod_price相同则按照prod_name进行升序排序
